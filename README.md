# Applicant Tracking System (ATS)

## Overview

The Applicant Tracking System (ATS) is a simple and responsive web-based application developed using HTML. The project is designed to demonstrate the basic workflow of resume screening and candidate evaluation through a clean and user-friendly interface.

This project focuses on front-end structure and layout design while providing an understanding of how modern ATS platforms function in recruitment processes.

---

## Objectives

- Create a simple ATS interface using HTML
- Demonstrate candidate screening workflow
- Design a clean and organized user interface
- Understand the structure of recruitment management systems

---

## Features

- Resume Submission Interface
- Candidate Information Form
- ATS Score Display Section
- Simple and Responsive Layout
- User-Friendly Interface Design
